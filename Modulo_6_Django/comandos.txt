cria uma local
mkdir noma_da_pasta

navega até a pasta com cd noma_da_pasta
cd noma_da_pasta

criar VM
python -m venv meu_projeto

ativar VM
scripts/activate

Se o comando acima der erro, execute este comando: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

instalar django
pip install django

criar projeto
django-admin startproject projeto1 .

comando para rodar o servidor
python manage.py runserver

detecta alterações nos modelos
python manage.py makemigrations

aplica as migrações no banco de dados
python manage.py migrate

comando para criar superusuario
python manage.py createsuperuser

criar app
python manage.py startapp nomedoap
