cria uma local
mkdir noma_da_pasta

navega até a pasta com cd noma_da_pasta
cd noma_da_pasta

criar VM
python -m venv venv

ativar VM
scripts/activate

Se o comando acima der erro, execute este comando: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

com o ambiente virtual ativo:

instalar django
pip install django

criar projeto
django-admin startproject projeto_final .

comando para rodar o servidor
python manage.py runserver

comando para parar o servidor
Control + C 

detecta alterações nos modelos
python manage.py makemigrations

aplica as migrações no banco de dados
python manage.py migrate

comando para criar superusuario
python manage.py createsuperuser

criar app
python manage.py startapp aluno

 path('meu_app/', include('meu_app.urls')),

url do app define qual view chamar
urlpatterns = [
    path('', home),
]

def home(request):
    return render(request,'index.html')
